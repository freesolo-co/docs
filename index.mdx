Flash is Freesolo's managed post-training service. You write a short config,
run one command, and Flash fine-tunes a model on a managed GPU — then serves the
result behind an OpenAI-compatible endpoint. There's nothing to host: every
command is a thin call to the hosted control plane, authenticated with your
Freesolo API key.

```bash
flash train config.toml      # fine-tune on a managed GPU
flash deploy <run-id>        # serve the trained adapter
flash chat <run-id> -m "hi"  # talk to it
```

## How it works

<Steps>
  <Step title="Pick a base model">
    Choose from the supported catalog (`flash models`). Flash trains a LoRA
    adapter on top — small, cheap, and fast to serve.
  </Step>
  <Step title="Point at an environment">
    An [environment](/guides/environments) is the task: the data your model sees
    and the rubric it's graded on. You author it as a small Python class (a
    Freesolo `EnvironmentSingleTurn`), publish it with `flash env push`, and
    reference it from your config by id.
  </Step>
  <Step title="Train on a managed GPU">
    `flash train` submits the run to the control plane, which allocates the
    cheapest GPU that fits (RunPod, with Lambda Labs as a capacity fallback),
    supervises the job (stall watchdog, bounded auto-retry from the last
    checkpoint), and streams logs back to you.
  </Step>
  <Step title="Deploy and use it">
    `flash deploy` serves the adapter. `flash chat` (or any OpenAI-compatible
    client) talks to it. Serving scales to zero (no charge while idle) and is
    billed per token.
  </Step>
</Steps>

## What you get

<CardGroup cols={2}>
  <Card title="SFT and GRPO" icon="brain">
    Supervised fine-tuning or reinforcement learning (GRPO), selected with one
    line in your config.
  </Card>
  <Card title="One GPU per run" icon="microchip">
    Each run gets a dedicated managed GPU, supervised server-side with bounded
    auto-retry that resumes from the last streamed checkpoint.
  </Card>
  <Card title="Cheapest GPU that fits" icon="dollar-sign">
    The allocator picks the lowest-cost GPU class that fits your run. See live
    pricing with `flash gpus`.
  </Card>
  <Card title="No infrastructure" icon="cloud">
    No GPUs to rent, no servers to run, no provider keys. You authenticate with
    a Freesolo API key; the platform owns the rest.
  </Card>
</CardGroup>
